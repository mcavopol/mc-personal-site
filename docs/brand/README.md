# Narrative Asset Library

Single source of truth for Michael Cavopol's brand messaging, bios, proof points, and content blocks. Any project — website, pitch deck, LinkedIn, conference bio, press kit — should pull from here.

## Files

| File | What it is | When to use it |
|------|-----------|----------------|
| [`messaging-framework.md`](./messaging-framework.md) | Strategic narrative, messaging house, brand voice, AI thesis | Establishing tone, writing new content, onboarding a designer or copywriter |
| [`bios.md`](./bios.md) | Pre-written bios at 5 lengths (15–500 words) | Speaker intros, conference programs, LinkedIn, bylines, press kits |
| [`proof-points.md`](./proof-points.md) | Roles, metrics, quotes, and companies — structured and tagged by pillar | Building case studies, selecting examples for a specific audience, populating data-driven sections |
| [`content-blocks.md`](./content-blocks.md) | Pre-written blocks tailored to 5 audiences | Customizing a pitch, writing an outreach email, tailoring a bio for context |
| [`data.json`](./data.json) | Machine-readable version of all roles, metrics, quotes, companies, and bios | Programmatic use — website components, templates, automation |

## Brand Voice Quick Reference

**Five attributes:** Direct, Warm, Confident, Team-centric, Specific

**Key rules:**
- First person, conversational ("We built..." not "Mr. Cavopol delivered...")
- "We" for results, "I" for decisions
- Numbers without commentary ($3M to $18M ARR — not "an astounding 6x growth")
- No empty adjectives (cut "innovative," "cutting-edge," "world-class")
- Short sentences. Active voice.
